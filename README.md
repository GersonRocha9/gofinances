## About

Application for personal financial management development in React Native.

---

### Clone Repo

```bash
$ git clone https://github.com/GersonRocha9/gofinances.git

$ cd Ignite_GoFinances
```

### Run

```bash
$ yarn install ou npm install

$ expo start
```

### .ENV Config

```bash

# Google Credentials
CLIENT_ID=
REDIRECT_URI=
RESPONSE_TYPE=
SCOPE=
```
